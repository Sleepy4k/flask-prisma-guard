Tentang Proyek

MindCheck SRQ-29 adalah alat skrining awal untuk membantu mahasiswa mengenali kemungkinan risiko kesehatan mental berdasarkan kombinasi data demografis dan jawaban kuesioner SRQ-29.

Metodologi

Sistem menggunakan alur pemodelan Machine Learning berbasis Logistic Regression untuk klasifikasi risiko, didahului oleh tahapan Standard Scaler dan Principal Component Analysis (PCA) agar fitur lebih stabil dan efisien saat inferensi.

33
Total Fitur Input (4 demografi + 29 SRQ)
2
Kelas Output (Aman / Perlu Perhatian)

Privasi Sistem

  • Tidak menggunakan database.
  • Tidak menyimpan riwayat input pengguna.
  • Pemrosesan dilakukan saat request berlangsung.
  • Setelah halaman ditutup, data tidak dipertahankan oleh aplikasi.

Batasan Sistem

Hasil dari aplikasi ini bukan diagnosis medis, melainkan skrining awal. Jika Anda atau teman Anda mengalami gejala yang mengganggu aktivitas harian, segera hubungi layanan konseling kampus, psikolog, atau tenaga kesehatan profesional.

Tentang Proyek

Prisma Guard adalah alat skrining awal untuk membantu mahasiswa mengenali kemungkinan risiko kesehatan mental berdasarkan kombinasi data demografis dan jawaban kuesioner SRQ-29.

Metodologi

Sistem menggunakan alur pemodelan Machine Learning berbasis Logistic Regression untuk klasifikasi risiko, didahului oleh tahapan Standard Scaler dan Principal Component Analysis (PCA) agar fitur lebih stabil dan efisien saat inferensi.

  • Pengumpulan data menggunakan kuesioner SRQ-29
  • Pra-pemrosesan data dan normalisasi fitur
  • Reduksi dimensi menggunakan Principal Component Analysis (PCA)
  • Klasifikasi risiko depresi menggunakan algoritma Logistic Regression
  • Penyajian hasil prediksi dan rekomendasi melalui sistem berbasis web

Batasan Sistem

  • Sistem hanya menghasilkan klasifikasi biner risiko depresi.
  • Sistem tidak menampilkan tingkat keparahan depresi.
  • Rekomendasi yang diberikan bersifat umum dan berdasarkan jawaban pengguna.
  • Hasil prediksi tidak dimaksudkan sebagai diagnosis medis.

Catatan Penting

Hasil dari aplikasi ini bukan diagnosis medis, melainkan skrining awal. Jika Anda atau teman Anda mengalami gejala yang mengganggu aktivitas harian, segera hubungi layanan konseling kampus, psikolog, atau tenaga kesehatan profesional.

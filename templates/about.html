Tentang Proyek

Prisma Guard adalah alat skrining awal yang dirancang untuk membantu mahasiswa dalam mengidentifikasi risiko depresi secara dini berdasarkan data demografis dan jawaban kuesioner SRQ-29 dengan memanfaatkan model machine learning.

Metodologi

Sistem ini menggunakan pendekatan teknologi machine learning dengan algoritma Logistic Regression untuk mengklasifikasikan risiko depresi. Sebelum proses klasifikasi, data terlebih dahulu melalui tahap standarisasi menggunakan Standard Scaler dan reduksi dimensi menggunakan Principal Component Analysis (PCA) agar proses klasifikasi lebih stabil dan akurat.

  • Pengumpulan data menggunakan kuesioner SRQ-29
  • Pra-pemrosesan data dan standarisasi fitur menggunakan StandardScaler
  • Reduksi dimensi menggunakan Principal Component Analysis (PCA)
  • Klasifikasi risiko depresi menggunakan algoritma Logistic Regression
  • Penyajian hasil prediksi dan rekomendasi melalui sistem berbasis web

Batasan Sistem

  • Sistem hanya menghasilkan klasifikasi risiko depresi dalam bentuk biner.
  • Sistem tidak menampilkan tingkat keparahan depresi.
  • Rekomendasi yang diberikan bersifat umum dan berdasarkan jawaban pengguna.
  • Hasil prediksi tidak dimaksudkan sebagai diagnosis medis.

Catatan Penting

Hasil dari aplikasi ini bukan merupakan diagnosis medis, melainkan hanya sebagai skrining awal. Jika Anda mengalami gejala yang mengganggu aktivitas sehari-hari, segera hubungi layanan konseling kampus, psikolog, atau tenaga kesehatan profesional.

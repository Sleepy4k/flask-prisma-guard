Mulai Prediksi

Kenapa MindCheck?

  • Proses cepat, sekitar 3-5 menit.
  • Input berbasis SRQ-29 + data demografis.
  • Hasil probabilitas disertai rekomendasi awal.
  • Tidak menyimpan data pengguna ke database.

03

Lihat Hasil Prediksi
